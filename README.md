# Deutlich Maps

> A TypeScript-based maps and geolocation application with AI-powered automation.

## About

Deutlich Maps is an intelligent mapping application built with TypeScript that provides geolocation services, map visualization, and automated data processing. The project combines modern web technologies with AI-assisted workflows to deliver a powerful mapping experience.

## Tech Stack

| Technology | Usage |
|------------|-------|
| TypeScript | Core application language (81.1%) |
| JavaScript | Scripting and tooling (9.5%) |
| Python | Backend automation and AI processing (7.0%) |
| CSS | Styling and layout (2.4%) |

## Project Structure

deutlich-maps/

├── shell_output_save/    # Shell output processing
├── user_input_files/     # User input handling
├── workspace.json        # Workspace configuration
├── pyproject.toml        # Python project configuration
├── deploy_url.txt        # Deployment URL reference
└── .gitignore            # Git ignore rules

## Features

- Interactive map visualization
- Geolocation services
- TypeScript-first architecture
- Python-powered backend processing
- Automated data download and processing
- Browser-compatible deployment
- Vercel deployment ready

## Deployment

The application is deployed on Vercel (Production). See deploy_url.txt for the live URL.

## Getting Started

1. Clone the repository:
   git clone https://github.com/fuatsevinc/deutlich-maps.git

2. Navigate into the project directory:
   cd deutlich-maps

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev
